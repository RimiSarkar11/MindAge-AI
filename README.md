# MindAge AI

MindAge AI is an NLP-powered web application that analyzes conversational text and estimates mental maturity indicators using sentiment analysis techniques.

The project uses Flask, Python, TextBlob, and VADER Sentiment Analysis to evaluate emotional tone, communication style, empathy indicators, and conversational behavior patterns.

---

## Features

- Conversational text analysis
- Estimated mental maturity age
- Positivity, negativity, and neutrality scoring
- Emotional stability analysis
- Empathy score detection
- Impulsiveness analysis
- Communication style prediction
- Interactive web interface using Flask

---

## Technologies Used

- Python
- Flask
- HTML
- CSS
- TextBlob
- VADER Sentiment Analysis
